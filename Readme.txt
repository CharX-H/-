1.第一次使用请先用记事本方式打开redit.bat文件修改注释处的自动启动时间  （默认启动时间为7：30）
然后双击运行 redit.bat 文件，来添加自动化启动功能 
******************************************************
**注意 自动启动须在电脑开机后才能运行
******************************************************

2.配置信息如学习通的账号密码，课程名字，请到set.ini文件中设置，
****************************************************
**注意课程名字只需包含关键词，关键词必须连续。
****************************************************

3.软件有2种启动方式（1）通过redit.bat注册计时器后可以每天定时启动 （2）手动运行 学习通签到.exe 文件
